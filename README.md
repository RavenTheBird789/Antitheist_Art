The file contained within this repository is a python program using ANSI colors to display an art piece showcasing an inverted cross, 
a pentacle, and a quote from be about religion and secularism

To run, simply type "git clone https://github.com/RavenTheBird789/Antitheist_Art" in your terminal and "python3 antitheist_art.py" and the art work will be displayed on your screen
